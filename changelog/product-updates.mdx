## Improvements

### xpander-mono
- Enhanced header-based authentication and secrets usage in the AI Gateway proxy, including consolidated Basic auth handling and improved security for forwarded headers.
- Outbound connector requests now consistently handle Basic Auth formatting and securely resolve credentials via AWS Secrets Manager, with optional role assumption support.

### frontend
- Updated texts in agno settings, set "Mask detected PII" default to true, and fixed metrics tab display.
- Refined agent metrics counting, enabled deep-linked agent creation via URL, and standardized terminology in the UI.
- Added built-in tools accordion in Agent Settings with toggleable tools, including Sleep. Improved progress visibility in thread logs and supported self-hosted chat URLs.
- Enabled deep linking from Tasks to the agent’s Activity view, focusing the specific thread for faster triage and debugging.
- Added OAuth2 CLI login flow with callback redirect, including API key selection and authorization UI.
