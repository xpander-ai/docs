**Dec 6 2025:**

**New Features:**
- Reasoning tools (Think/Analyze) with WebUI support showing reasoning steps in conversations
- Agent activity threads API with Redis storage for comprehensive viewing of agent messages, tool calls, and reasoning
- Tool calls compression with configurable settings and thresholds
- Agentic memory & culture feature toggles for enhanced agent capabilities
- New LLM providers: Google AI Studio (Gemini 2.0-flash, 2.5-pro, 3-pro-preview), Helicone, OpenRouter, Nebius Token Factory
- Expanded framework support: Strands Agents, Google ADK, OpenAI Agents, LangChain with appropriate feature gating

**Improvements:**
- Pre-started dual streams (SDK+worker) for reduced latency in agent responses
- Performance enhancements (PRO-442) across platform
- Searchable LLM modal with improved filtering capabilities
- Non-blocking K8s metrics ThreadPoolExecutor for better scalability
- Async history upsert for improved data handling
- Reasoning and tool call compression enabled by default for new agents
- Enhanced framework selection with per-framework feature gating

**Bug Fixes:**
- iOS/Safari chat input textarea height responsiveness
- TaskFinished signaling with proper terminal status handling
- iOS OAuth2 link handling and loading/disabled states
- Framework icon and name display in settings
- Failed task status now properly treated as terminal with finished_at timestamp
- Hardened connector headers and security dictionary validation
