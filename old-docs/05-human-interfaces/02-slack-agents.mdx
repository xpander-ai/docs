Slack Agents let you bring the power of xpander.ai's AI Agents directly into any Slack workspace. With just a few steps, you can create, connect, and customize intelligent agents that automate tasks, answer questions, and boost productivity - all from within Slack. Learn more at [slack.xpander.ai](https://slack.xpander.ai/).

<Tip>
You can build new agents or connect existing xpander.ai agents to Slack, enabling advanced workflows and real-time collaboration.
</Tip>

## Key Features & Benefits

<CardGroup>
  <Card title="Slack Agent" icon="slack">
    An agent entity that is connected to a Slack workspace of your choice. You can then connect it to a Custom AI Agent on the xpander platform, as well as configure it to auto-engage in channels on specific topics, add conversation starters, and more.
  </Card>
  <Card title="Custom AI Agent" icon="robot">
    The agent entity that the Slack Agent entity connects to. These are AI agents running on the xpander.ai Agent platform, ready for you to build, customize, test, and easily connect to Slack.
  </Card>
  <Card title="Zero Infrastructure" icon="server">
    Add Custom AI Agents to any Slack Workspace without the development and infrastructure overhead. xpander.ai handles hosting and stitches the integration directly to your custom AI Agents.
  </Card>
  <Card title="Versatile Use-Cases" icon="users">
    Perfect for internal use, joint customer channels, and community support - add your Slack Agent to any Slack Workspace. Automate tasks like opening support tickets, scheduling calls, and more - whatever you can build with a Custom AI Agent.
  </Card>
</CardGroup>

## Setup Guide

<Tabs>
  <Tab title="Start from Scratch">
    <Steps>
      <Step title="Go to Slack Agents section">
        In the xpander.ai console, navigate to the **Slack Agents** section.
        {/* Screenshot: Slack Agents section in xpander.ai UI */}
      </Step>
      <Step title="Create your Slack AI Agent">
        Click **Create Agent** and follow the prompts to set up your new Slack AI Agent.
        {/* Screenshot: Create Slack Agent form */}
      </Step>
      <Step title="Connect to your xpander AI Agent">
        Link your Slack Agent to an existing xpander AI Agent, or create a new one if needed.
        {/* Screenshot: Agent connection panel */}
      </Step>
      <Step title="Customize or build your agent">
        You can further customize your agent's behavior, or build a new agent from scratch in xpander.ai for advanced use cases.
        {/* Screenshot: Agent customization options */}
      </Step>
      <Step title="Start using your Slack Agent">
        Once connected, your Slack Agent is ready to use in your workspace. Interact with it in channels or direct messages to automate tasks and get instant answers.
        {/* Screenshot: Slack conversation with AI Agent */}
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Deploy Template">
    <Steps>
      <Step title="Browse Agent Templates">
        Go to the [xpander.ai Agent Hub](https://slack.xpander.ai/templates/) to find a list of ready-to-use AI Agents.
        {/* Screenshot: xpander.ai Agent Hub page */}
      </Step>
      <Step title="Select and Deploy">
        Choose an agent that fits your needs and click "Deploy". This will redirect you to Slack to authorize the installation.
        {/* Screenshot: Slack authorization screen */}
      </Step>
      <Step title="Automatic Setup">
        Once authorized, xpander.ai automatically deploys a Custom AI Agent template to your account and connects it to your new Slack Agent.
        {/* Screenshot: Deployed agent template in xpander.ai UI */}
      </Step>
      <Step title="Start Using and Customize">
        Your new agent is ready to use in Slack! You can further customize the underlying Custom AI Agent template from the xpander.ai console.
        {/* Screenshot: Agent interacting in a Slack channel */}
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Advanced features of Slack AI Agents in xpander.ai

<CardGroup cols={2}>
  <Card title="Auto OCR from Images" icon="image">
    Automatically read text from images (supports .jpg, .jpeg, .png, .gif, .bmp, .tiff, .tif, .webp, .heic, .avif)
  </Card>
  <Card title="Auto Transcribe Audio" icon="microphone">
    Automatically transcribe audio messages so the agent can process them
  </Card>
  <Card title="Auto OCR from PDF" icon="file-pdf">
    Automatically read text from PDF files
  </Card>
  <Card title="Auto Crawl Links" icon="link">
    Automatically crawl and fetch data from the first page of links sent to the agent
  </Card>
  <Card title="Auto-Engage in Channels" icon="hashtag">
    Define conditions when the agent should automatically engage in channels. For example: "When user is asking about stock price"
  </Card>
  <Card title="Conversation Starters" icon="sparkles">
    Create prompt suggestions for users. They will be presented on the chat UI before the first interaction of every new conversation.
  </Card>
</CardGroup>
