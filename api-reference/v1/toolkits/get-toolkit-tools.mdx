---
title: "Get Toolkit Tools"
openapi: "GET /v1/toolkits/{toolkit_id}/tools"
description: "Returns toolkit tools in tool calling structure"
---

Retrieve all tools within a specific toolkit, formatted in the tool calling structure compatible with AI model function calling.

## Path Parameters

<ParamField path="toolkit_id" type="string" required>
  Unique identifier of the toolkit (UUID format)
</ParamField>

## Response

Returns an array of tool definitions in OpenAI function calling format:

<ResponseField name="name" type="string">
  Tool function name
</ResponseField>

<ResponseField name="description" type="string">
  Description of what the tool does
</ResponseField>

<ResponseField name="parameters" type="object">
  JSON Schema defining the tool's input parameters
  <Expandable title="Parameters Schema">
    <ResponseField name="type" type="string">
      Always "object" for tool parameters
    </ResponseField>
    <ResponseField name="properties" type="object">
      Object defining each parameter's schema
    </ResponseField>
    <ResponseField name="required" type="array">
      Array of required parameter names
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Response

```json
[
  {
    "name": "search_web",
    "description": "Search the web for information",
    "parameters": {

      "properties": {

        "max_results": {
```
