---
title: "Create Agent"
openapi: "POST /v1/agents"
description: "Create a new AI agent with specified configuration, tools, and knowledge bases"
---

Create a new AI agent with custom configuration. The agent will be created and ready for deployment.

## Request Body

<ParamField body="name" type="string" required>
  Display name for the agent
</ParamField>

<ParamField body="deployment_type" type="string">
  Deployment infrastructure: `serverless` (default) or `container`
</ParamField>

<ParamField body="type" type="string">
  Agent type: `manager` (default)
</ParamField>

<ParamField body="environment_id" type="string">
  Target environment ID (optional)
</ParamField>

<ParamField body="using_nemo" type="boolean">
  Use Nvidia NeMo (default: false)
</ParamField>

## Response

Returns a complete `AIAgent` object:

<ResponseField name="id" type="string">
  Unique identifier for the created agent (UUID)
</ResponseField>

<ResponseField name="name" type="string">
  Display name of the agent
</ResponseField>

<ResponseField name="status" type="string">
  Current deployment status: `ACTIVE` or `INACTIVE`
</ResponseField>

<ResponseField name="deployment_type" type="string">
  Deployment infrastructure: `serverless`, `container`, or `null`
</ResponseField>

<ResponseField name="organization_id" type="string">
  UUID of the organization that owns this agent
</ResponseField>

<ResponseField name="icon" type="string">
  Emoji icon representing the agent
</ResponseField>

<ResponseField name="instructions" type="object">
  System instructions configuration
  <Expandable title="Instructions Object">
    <ResponseField name="role" type="array">
      Array of role descriptions
    </ResponseField>
    <ResponseField name="goal" type="array">
      Array of goal descriptions
    </ResponseField>
    <ResponseField name="general" type="string">
      General instructions text
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="model_provider" type="string">
  AI model provider (e.g., `openai`)
</ResponseField>

<ResponseField name="model_name" type="string">
  Specific model version (e.g., `gpt-4o`, `gpt-4.1`)
</ResponseField>

<ResponseField name="framework" type="string">
  Agent framework used (e.g., `agno`)
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of when the agent was created
</ResponseField>

## Example Request

```bash
curl -X POST -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My API Agent",
    "deployment_type": "serverless"
  }' \
  https://api.xpander.ai/v1/agents
```
