---
title: "Update Agent"
openapi: "PATCH /v1/agents/{agent_id}"
description: "Update an existing AI agent's configuration, tools, or knowledge bases"
---

Modify an agent's configuration. Only provided fields will be updated. At least one field must be provided.

## Path Parameters

<ParamField path="agent_id" type="string" required>
  Unique identifier of the agent to update (UUID format)
</ParamField>

## Request Body

<ParamField body="name" type="string">
  Display name for the agent
</ParamField>

<ParamField body="instructions" type="object">
  System instructions configuration
  <Expandable title="Instructions Object">
    <ParamField body="role" type="array">
      Array of role descriptions
    </ParamField>
    <ParamField body="goal" type="array">
      Array of goal descriptions
    </ParamField>
    <ParamField body="general" type="string">
      General instructions text
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="icon" type="string">
  Emoji icon representing the agent
</ParamField>

<ParamField body="model_provider" type="string">
  AI model provider (e.g., `openai`)
</ParamField>

<ParamField body="model_name" type="string">
  Specific model version (e.g., `gpt-4o`, `gpt-4.1`, `gpt-4o-mini`)
</ParamField>

<ParamField body="should_reset_graph" type="boolean">
  Reset the agent's graph (default: false)
</ParamField>

<ParamField body="environment_id" type="string">
  Target environment ID
</ParamField>

## Response

Returns the updated `AIAgent` object with all current configuration.

## Example Request

```bash
curl -X PATCH -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Updated Agent Name",
    "instructions": {
      "role": ["You are a helpful assistant"],
      "goal": ["Help users with their questions"],
      "general": "Be concise and accurate"
    },
    "model_name": "gpt-4o"
  }' \
  https://api.xpander.ai/v1/agents/{agent_id}
```

## Notes

- At least one field must be provided in the request body
- Only the specified fields will be updated
- Other fields remain unchanged
- The agent may need to be redeployed after significant changes
