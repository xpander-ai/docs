---
title: "Get Agent"
openapi: "GET /v1/agents/{agent_id}"
description: "Retrieve detailed information about a specific AI agent by its unique identifier"
---

Get complete details about an agent including configuration, deployment status, and metadata.

## Path Parameters

<ParamField path="agent_id" type="string" required>
  Unique identifier of the agent to retrieve (UUID format)
</ParamField>

## Common Use Cases

### Extracting Agent Configuration Details

Get comprehensive agent information including tools, settings, and webhook URLs:

<CodeGroup>
```python Python
import requests

API_KEY = 'your-api-key'
BASE_URL = 'https://api.xpander.ai/v1'

def get_agent_config(agent_id: str):
  response = requests.get(
    f'{BASE_URL}/agents/{agent_id}',
    headers={'x-api-key': API_KEY}
  )

  agent = response.json()

  # Extract key configuration
  config = {
    'name': agent['name'],
    'status': agent['status'],
    'webhook_url': agent['webhook_url'],
    'model': f"{agent['model_provider']}/{agent['model_name']}",
    'framework': agent['framework'],
    'version': agent['version'],
    'tools_count': len(agent['tools']),
    'deployment': agent['deployment_type']
  }

  return config

# Example usage
config = get_agent_config('3496ccfc-4547-4142-8fc2-b35c9a1c201d')
print(config)
```

```javascript JavaScript
const API_KEY = 'your-api-key';
const BASE_URL = 'https://api.xpander.ai/v1';

async function getAgentConfig(agentId) {
  const response = await fetch(`${BASE_URL}/agents/${agentId}`, {
    headers: { 'x-api-key': API_KEY }
  });

  const agent = await response.json();

  // Extract key configuration
  const config = {
    name: agent.name,
    status: agent.status,
    webhook_url: agent.webhook_url,
    model: `${agent.model_provider}/${agent.model_name}`,
    framework: agent.framework,
    version: agent.version,
    tools_count: agent.tools.length,
    deployment: agent.deployment_type
  };

  return config;
}

// Example usage
const config = await getAgentConfig('3496ccfc-4547-4142-8fc2-b35c9a1c201d');
console.log(config);
```
