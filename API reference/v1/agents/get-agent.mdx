---
title: "Get Agent"
openapi: "GET /v1/agents/{agent_id}"
description: "Retrieve detailed information about a specific AI agent by its unique identifier"
---

Get complete details about an agent including configuration, deployment status, and metadata.

## Path Parameters

<ParamField path="agent_id" type="string" required>
  Unique identifier of the agent to retrieve (UUID format)
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the agent (UUID)
</ResponseField>

<ResponseField name="name" type="string">
  Display name of the agent
</ResponseField>

<ResponseField name="icon" type="string">
  Emoji icon representing the agent
</ResponseField>

<ResponseField name="status" type="string">
  Current deployment status: `ACTIVE` or `INACTIVE`
</ResponseField>

<ResponseField name="organization_id" type="string">
  UUID of the organization that owns this agent
</ResponseField>

<ResponseField name="deployment_type" type="string">
  Deployment infrastructure type: `serverless`, `container`, or `null`
</ResponseField>

<ResponseField name="instructions" type="object">
  System instructions configuration
  <Expandable title="Instructions Object">
    <ResponseField name="role" type="array">
      Array of role descriptions
    </ResponseField>
    <ResponseField name="goal" type="array">
      Array of goal descriptions
    </ResponseField>
    <ResponseField name="general" type="string">
      General instructions text
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="model_provider" type="string">
  AI model provider (e.g., `openai`)
</ResponseField>

<ResponseField name="model_name" type="string">
  Specific model version (e.g., `gpt-4o`, `gpt-4.1`, `gpt-4o-mini`)
</ResponseField>

<ResponseField name="framework" type="string">
  Agent framework used (e.g., `agno`)
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of when the agent was created
</ResponseField>

<ResponseField name="using_nemo" type="boolean">
  Whether the agent uses Nvidia NeMo
</ResponseField>

<ResponseField name="is_coordinate_mode" type="boolean">
  Whether the agent operates in coordinate mode
</ResponseField>

## Example Request

```bash
curl -X GET -H "x-api-key: YOUR_API_KEY" \
  https://api.xpander.ai/v1/agents/2c126a59-b625-4e4a-a008-cb3eb76cc3b2
```
