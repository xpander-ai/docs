---
title: "Invoke Tool"
openapi: "POST /v1/toolkits/{toolkit_id}/tools/{tool_id}"
description: "Invokes a specific tool in a toolkit"
---

Execute a specific tool within a toolkit with provided parameters. This allows direct tool invocation outside of agent execution.

## Path Parameters

<ParamField path="toolkit_id" type="string" required>
  Unique identifier of the toolkit (UUID format)
</ParamField>

<ParamField path="tool_id" type="string" required>
  Identifier of the specific tool to invoke
</ParamField>

## Request Body

<ParamField body="parameters" type="object" required>
  Tool-specific parameters matching the tool's schema from [Get Toolkit Tools](/api-reference-rest/toolkits/get-toolkit-tools)
</ParamField>

## Response

<ResponseField name="result" type="any">
  The tool execution result (format varies by tool)
</ResponseField>

<ResponseField name="status" type="string">
  Execution status: `success`, `error`
</ResponseField>

<ResponseField name="execution_time" type="number">
  Tool execution time in seconds
</ResponseField>

## Example Request

```bash
curl -X POST -H "x-api-key: YOUR_API_KEY" \
  https://api.xpander.ai/v1/toolkits/{toolkit_id}/tools/search_web \
  -d '{
    "parameters": {
      "query": "AI agent frameworks",
      "max_results": 5
    }
  }'
```

## Example Response

```json
{
  "result": [
    {
      "title": "Top AI Agent Frameworks",
      "url": "https://example.com/article",
      "snippet": "Comprehensive guide to AI agent frameworks..."
    }
  ],
  "status": "success",
  "execution_time": 1.23
}
```
