## Two Ways to Build AI Agents

xpander.ai supports two distinct approaches depending on your use case complexity and requirements.

  <Card title="Autonomous Agents with a Specific Role" icon="robot" href="#1-autonomous-agents-with-a-specific-role-low-code--no-code">

## 1. Autonomous Agents with a Specific Role (Low Code / No Code)

Production-ready autonomous agents via Visual Workbench - no code required. The platform manages the Agno runtime for you.

<CardGroup cols={2}>
  <Card title="Key Characteristics" icon="list-check">
    - Clearly defined agent role and goals
    - Non-deterministic behavior OK
    - Single or multi-agent with tools
    - Configuration driven
  </Card>
  <Card title="Example Use Cases" icon="lightbulb">
    - Website chatbots and support
    - DevOps and IT helpdesk assistants
    - PR review, log summarization
    - Sales qualification bots
  </Card>
</CardGroup>

**Try it:** [xpander.ai website agent](https://xpander.ai)

---

## 2. Complex Multi-Framework Applications (Code + Containers)

Same xpander infrastructure, but you write the code. Use your preferred framework with custom logic.

<CardGroup cols={2}>
  <Card title="Key Characteristics" icon="list-check">
    - Custom code and logic
    - Deterministic behavior when needed
    - Any framework (LangChain, Agno, etc.)
    - Container-based deployment
  </Card>
  <Card title="Example Use Cases" icon="lightbulb">
    - RAG-powered SaaS products
    - Trading and analytics platforms
    - Healthcare and compliance apps
    - Document processing systems
  </Card>
</CardGroup>
