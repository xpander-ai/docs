## No-Code Agent Builder

Build and deploy production-ready agents directly in the UI without writing code. Select tools, edit the system prompt, enable memories and other advanced features - the platform manages the Agno runtime for you.

<Info>
More frameworks will be added to the no-code offering soon. [Join the beta](https://xpander.ai) to get early access.
</Info>

<Steps>
  <Step title="Create your agent">
    Use the Workbench at [app.xpander.ai](https://app.xpander.ai) to set up your agent's instructions, model, tools, and knowledge bases through the intuitive web interface.

    <Frame caption="The Agent Workbench - configure your agent without code">
    ![Agent Workbench](/images/workbench.png)
    </Frame>
  </Step>

  <Step title="Configure the backend">
    Set up your agent's capabilities:

    - **AI Model** - Choose from OpenAI, Anthropic, or bring your own keys
    - **System Prompt** - Define your agent's role, personality, and behavior
    - **Tools** - Add connectors from the catalog (Slack, GitHub, Jira, etc.)
    - **Knowledge Base** - Upload documents for RAG capabilities
    - **Memory** - Enable conversation history and user context
  </Step>

  <Step title="Connect task sources">
    Choose how users will interact with your agent:

    - **Slack** - Deploy directly to your workspace
    - **Web UI** - Get a hosted chat interface
    - **Webhooks** - Trigger from external systems
    - **API** - Invoke programmatically
    - **Scheduled Tasks** - Run on a schedule
  </Step>

  <Step title="Test and deploy">
    Test your agent in the Workbench chat, then deploy with one click. Your agent is live with automatic scaling and monitoring.
  </Step>
</Steps>

### Using the API

You can also create and invoke no-code agents programmatically:

<CodeGroup>

```bash Create Agent
curl --request POST \
  --url https://api.xpander.ai/v1/agents \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <your-api-key>' \
  --data '{
  "name": "my-cool-agent",
  "deployment_type": "serverless",
  "framework": "agno"
}'
```

```bash Invoke Agent
curl --request POST \
  --url https://api.xpander.ai/v1/agents/<agent-id>/invoke \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <your-api-key>' \
  --data '{
    "input": {
      "text": "hi!"
    }
  }'
```

```bash Async Task
curl --request POST \
  --url https://api.xpander.ai/v1/agents/<agent-id>/invoke/async \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <your-api-key>' \
  --data '{
    "input": {
      "text": "Generate a weekly report"
    }
  }'
```

```bash List Tasks
curl --request GET \
  --url 'https://api.xpander.ai/v1/tasks?agent_id=<agent-id>' \
  --header 'x-api-key: <your-api-key>'
```

```bash Create Knowledge Base
curl --request POST \
  --url https://api.xpander.ai/v1/knowledge \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <your-api-key>' \
  --data '{
  "name": "my-agent-kb",
  "description": "Knowledge base for my agent"
}'
```

</CodeGroup>

## Code + Containers

If you need custom logic, use multiple frameworks, or require deterministic behavior:

```bash
# Create a new agent and select a framework template
xpander agent new
```

Available templates:

- **Agno** - The fastest agent framework
- **Strands** - AWS Strands integration
- **Google ADK** - Google's Agent Development Kit
- **OpenAI Agents SDK** - OpenAI's agent framework
- **LangChain** - Popular LLM orchestration framework

Or follow the [Frameworks Guide](/user-guide/frameworks) to instrument your existing code with xpander.
